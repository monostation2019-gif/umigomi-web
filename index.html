# 海ゴミ探究ノート（HTML/JS版）v1.1 ターミナルもインストール作業も不要です。ファイルを編集して、Vercelにアップロードするだけで動きます。 ## 構成 ``` umigomi_web/ ├── index.html ← アプリ本体（これ1つに画面・処理すべてが入っています） ├── manifest.json ← PWA設定 ├── sw.js ← オフライン対応（そのままでOK、触らなくて大丈夫） ├── favicon.png └── icons/ ← アプリアイコン・初期プロフィール画像 ``` ## v1.1で実装済みの機能 - ログイン（Googleログイン／ゲスト（匿名）ログイン）・自動ログイン - ホーム画面（歓迎メッセージ・記録件数・ショートカット・使い方） - プロフィール画面（ニックネーム・ひとこと・画像変更） - **地図画面**：みんなが記録したゴミをピンで表示（OpenStreetMap + Leaflet.js） - **記録一覧画面**：みんなの記録を新しい順に一覧表示 - **記録の追加**：写真（必須）・種類（6カテゴリ）・場所（GPS自動取得＋地図タップで手動修正）・ひとことメモ（任意）を記録 - 記録の詳細表示（一覧・地図のピンどちらからも開ける） - 藍色ヘッダー・ボトムナビのデザイン すべての記録は**全ユーザーで共有**され、誰でも地図・一覧で見られます（投稿者名も表示されます）。 ## セットアップ手順 ### ① Firebaseの接続情報を書き込む 1. `index.html` を **メモ帳（またはVSCode）** で開く 2. `Ctrl + F` で `REPLACE_ME` を検索する（6箇所あります） 3. Firebaseコンソール（https://console.firebase.google.com）を開き、 - 左上の**⚙️歯車アイコン → プロジェクトの設定**をクリック - 下の方の「マイアプリ」に、既にWebアプリが登録されていればそれをクリック （無ければ `` アイコンをクリックして新規登録） - 表示される `firebaseConfig = { ... }` の中身をまるごとコピー 4. `index.html` 内の該当部分を、コピーした内容で置き換えて保存する ### ② Firebaseコンソールで設定する項目 - **Authentication → Sign-in method** - 「匿名」を有効化 - 「Google」を有効化（Googleログインを使う場合） - **Firestore Database** を作成する（無ければ「データベースを作成」） - **Firestore Database → ルール** タブで、以下を貼り付けて「公開」 ``` rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{uid} { allow read, write: if request.auth != null && request.auth.uid == uid; } match /records/{recordId} { allow read: if true; allow create: if request.auth != null && request.auth.uid == request.resource.data.uid; allow update, delete: if request.auth != null && request.auth.uid == resource.data.uid; } } } ``` > Firebase Storageは使用していないため、設定不要です。 ## 動作確認の方法 **方法A（一番簡単）：そのままダブルクリックで開く** `index.html` をダブルクリックしてブラウザで開くだけで動作確認できます。 （GPSの現在地取得は `file://` でも動きますが、うまくいかない場合は方法Bをお試しください） **方法B（本番と同じ環境で確認したい場合）：Vercelにアップロード** `umigomi_web` フォルダを丸ごとVercelにドラッグ＆ドロップしてください。 ## 動かない時のチェックリスト | 症状 | 原因の可能性 | |---|---| | ずっと「準備中…」のまま | `REPLACE_ME` が残っている / Firebaseの値が間違っている | | 赤い文字でエラーが出る | エラー文をそのままコピーして質問してください | | ログインできない | Authentication の「匿名」または「Google」が無効になっている | | プロフィール・記録が表示されない／権限エラー | Firestore Database未作成、またはルール未設定（`records`のルールも追加されているか確認） | | 地図が真っ白・崩れて見える | インターネット接続を確認（地図タイルの読み込みに通信が必要です） | | 現在地が取得できない | ブラウザの位置情報許可がブロックされていないか確認。地図をタップして手動で場所を指定することもできます | ## 今後の拡張予定（v1.2以降） - カテゴリの絞り込み表示（地図・一覧） - 自分の記録だけの絞り込み表示 - 記録の削除・編集 - 13カテゴリへの拡張 ## 変更履歴 - **v1.1**：地図・記録一覧・記録の追加（写真＋カテゴリ＋位置情報＋メモ）を実装 - **v1.0**：ゲスト/Googleログイン、プロフィール（画像変更含む）、ホーム画面、ボトムナビの土台が完成
